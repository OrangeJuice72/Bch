# Blankenship Custom Homes Website

A modern React + Vite website starter for Blankenship Custom Homes.

## Pages included

- Home
- Portfolio
- Current Builds
- Contact Us

## Features

- React Router navigation
- Full-width hero section
- Professional modern layout
- Placeholder imagery ready to replace with your real assets
- Responsive design for desktop, tablet, and mobile

## Getting started

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Notes

- Placeholder images currently use `picsum.photos`
- The contact form is UI-only right now and not connected to a backend service yet
- You can replace the placeholder contact info, address, and project details in the page components

## Suggested next steps

1. Replace placeholder images with your branded photography/assets
2. Add your real contact details
3. Connect the form to your preferred form handler/backend
